# Project Description

This project is a personal portfolio website developed to present creative and academic work across different disciplines, such as:

- Graphic design
- Visual communication
- Editorial and branding projects
- Digital and multimedia design

The website follows a clean and minimalist design approach, focusing on clarity, balance, and usability.  
Special attention has been given to layout organization, typography, and responsive behavior across devices.

==============================

## Purpose

- Showcase selected creative projects in a professional format
- Present personal profile, skills, and background
- Practice and apply front-end web development skills
- Create a clear and navigable portfolio website
- Serve as a final academic project

==============================

## Tech Stack

### Core Technologies

- **HTML5** — Semantic and structured markup
- **CSS3** — Styling, layout, and responsive design
- **JavaScript (Vanilla)** — Basic interactivity and dynamic behavior

### Technical Features

- Responsive design (desktop, tablet, and mobile)
- Flexbox and CSS Grid layouts
- Smooth navigation between sections
- Clean and accessible structure
- Organized and reusable CSS styles

### Hosting

- **GitHub Pages** — Static site deployment

# Local Setup Instructions

## Prerequisites

- A modern web browser
- Visual Studio Code
- Live Server extension (recommended)

## Installation

Clone the repository:

```bash
git clone https://github.com/irenesansa/FINAL-PROJECT-PORTFOLIO-TEMPLATE.git
cd FINAL-PROJECT-PORTFOLIO-TEMPLATE
Run the project locally:

Option A — VS Code
Install the Live Server extension
Right-click index.html → Open with Live Server

Option B — Open directly
Open index.html in your browser

==============================

File Structure
FINAL-PROJECT-PORTFOLIO-TEMPLATE/
├── index.html # Main homepage
├── assets/
│ ├── css/ # Stylesheets
│ ├── js/ # JavaScript files
│ └── images/ # Project images
└── README.md # Project documentation

==============================

Customization Guide
Content
Update personal information in the About section

Replace project images and descriptions

Modify text content directly in index.html

Styles
Edit colors, typography, and layout in the CSS files

Adjust spacing and responsive behavior as needed

Projects
Add or remove project sections

Update links and images for each project

Maintain a consistent visual structure

==============================

Key Features
Clean and minimalist visual design

Fully responsive layout

Easy navigation between sections

Clear presentation of creative projects

Structured and readable codebase

==============================

Deployment
This project is deployed using GitHub Pages.

Deployment steps:

Push the repository to GitHub

Go to Settings → Pages

Select branch main and /root

Save and wait for deployment
```
